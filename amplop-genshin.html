Amplop Lebaran Tema Genshin Impact

Oleh: Ceight Ch.

Halo Assalaamu'alaikum Warahmatullahi Wabarakaatuh.

Pada kesempatan kali ini, saya ingin membagikan hasil kreasi desain CorelDraw berupa Amplop Lebaran dengan tema Genshin Impact. Tema ini saya pilih karena saya sendiri merupakan pemain game tersebut, sehingga terinspirasi untuk membuat desain amplop yang unik dan berbeda dari biasanya.

Desain amplop ini menggunakan ukuran 7x13 cm, agar tata letaknya pas saat dicetak di kertas A3+ / A3 Extension sebanyak 6 pcs. Ukuran tersebut menurut saya cukup ideal, tidak terlalu kecil maupun besar. Terdapat dua model desain yang bisa dipilih, dan jika ingin melakukan kustomisasi, saya juga membagikan file CorelDraw-nya agar dapat diedit sesuai kebutuhan.

Untuk mengunduh file desain, silakan klik tombol di bawah artikel ini. Sebelum membuka file di CorelDRAW (minimal versi X7), sangat disarankan untuk menginstal font yang digunakan agar tampilan tulisan tidak error.

Demikian sharing dari saya, mohon maaf apabila terdapat kekurangan atau kesalahan. Terima kasih telah membaca.
