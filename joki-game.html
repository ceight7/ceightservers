Joki Akun Game

Oleh: Ceight Ch.

Halo, Assalaamu’alaikum Warahmatullahi Wabarakaatuh. Ini adalah sedikit penjelasan QnA buat kalian yang ingin joki akun game ke saya. Saya mohon dibaca dengan seksama karena ada hal yang penting yang saya sampaikan untuk ke kalian.

Terkadang sebelumnya saya pernah ngestream di youtube tapi dengan kondisi tidak ada konten ataupun yang dikerjain. Sehingga saya bingung mau ngapain karena yang dikerjakan sudah selesai semua. Kadang ada penonton di live chat request joki akun dan saya senang mendengar itu karena saya jadi ada kegiatan.

Game yang sekarang saya mainkan adalah Honkai Impact 3rd sama Genshin Impact, selain itu saya tidak main dengan alasan keterbatasan waktu dan perangkat. Jadinya hanya game yang saya mainkan yang bisa saya jokiin. Untuk yang Honkai Impact 3rd lebih lanjutnya bisa langsung lewat DM saja.

Untuk yang bisa saya jokiin adalah sebagai berikut Genshin Impact :

  • Daily Quest & Expedition
  • Weekly Quest
  • Daily Resin/Conden
  • Farming bahan
  • Eksplorasi Area
  • Spiral Abyss & Imaginarium Theater (?)
  • Event (?)

ada beberapa hal catatan yang saya tandai tanda tanya. Untuk Spiral Abyss dan Imaginarium Theater sebenarnya saya masih skill issue, tapi kalau maksa saya tetap kerjakan dengan catatan tergantung build karakter dan musuh yang dilawan. Dan saya sangat tidak bisa menjamin untuk floor 11 dan 12 bisa full star/babak 10 full star bahkan arcana. Untuk event tergantung model event yang di sediakan. Terkadang ada event yang saya sulit untuk mengerjakan seperti event perebutan name card terbatas dengan melawan bos yang termasuk sangat sulit. Untuk selebihnya bisa ditanyakan lanjut lewat DM saja.

Untuk pengerjaan saya lebih fokus ke malam hari. Untuk malam minggu saya libur karena saya juga ingin refreshing keluar rumah (biasanya saya main ke rumah temen buat nongkrong). Apakah bakal dikerjain juga pas livestream? Iya, dan juga kemungkinan saya show off ke livestream kalau saya pas livestream waktu senggang dan benar-benar gabut buat jadi bahan konten. Kalau pas lagi fokus story saya nggak ngerjain karena kemungkinan besar saya ketinggalan beberapa hal di gamenya.

Untuk hal itu saya pribadi lebih fokus untuk menjaga privasi. Alhasil untuk pas lagi livestream saya kasih kotak hitam kecil untuk menutupi uid, untuk nanti saat saya buka menu dan saat login saya buat blackscreen. Dan lebih demi mengutamakan privasi yang lebih mendalam saya memutuskan DM ke Discord saya atau ke Telegram saya, sehingga saat saya selesai saya akan menghapus seluruh chat di tempat saya (kalau ada menu hapus semua sampai menyeluruh diutamakan) dan saya sangat sarankan untuk hapus authorize device saya yang sudah tersambung di akun hoyoverse. Seandainya nanti privasi kecil bocor seperti kespill uid pas livestream nanti saya konfirmasi ke anda apakah perlu saya cut video atau tidak.

So, saya dari awal memang niatan untuk biaya joki saya free alias gratis. Mungkin ini keputusan yang ceroboh bahkan distract sama penjoki lain seperti menghancurkan pasar, tapi saya diri sendiri memang niatan menggratiskannya dan kalau saya memberi tarif berapa saya malah tidak terlalu pintar memberikan harga. Tapi, kalau ada dari kalian memang niatnya ngasih dan ngasihnya seikhlasnya bisa kirim ke donation saya. Saya sangat berterima kasih kalau ada yang memberi bantuan ke saya untuk kehidupan saya. Semoga apa yang saya berikan dan kerjakan menjadi berkah untuk kalian semua.

Kalian bisa DM saya ke Discord atau Telegram saya dengan nickname @ceight7. Saya tidak menerima dm lewat selain kedua social media atau komunikasi tersebut karena saya mengutamakan privasi baik dari saya maupun kalian. Apakah kalian mikir nggak berani dm karena bio saya agak ngedistract ? selama dm kalian memang urgensi dan fokus ke joki saya tidak akan ban/block dan saya tetap membacanya. Saya membuat bio tersebut dengan alasan banyak orang luar DM saya yang tidak penting dan menjadi spam sehingga saya ban bahkan block.

Itu saja yang saya sampaikan untuk yang minat saya joki akunnya. Semoga paham dengan kondisi dan ketentuan yang saya sampaikan. Terima kasih perhatiannya dan mohon maaf kalau ada salah dari kata-kata saya.

Wassalaamu’alaikum warahmatullahi wabarakaatuh.

Salam, Ceight.
